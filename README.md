# Flag Quiz Masters (世界の国旗クイズ)

モダンなデザインと使いやすさを追求した、Webブラウザで遊べる国旗クイズアプリです。

## 特徴
- **全世界の国旗に対応**: REST Countries APIを使用して最新のデータを取得。
- **日本語対応**: 国名やインターフェースはすべて日本語。
- **レスポンシブデザイン**: PCでもスマホでも快適にプレイ可能。
- **ハイスコア機能**: 自己ベストが自動的に保存されます。

## 遊び方
1. `index.html` をブラウザで開きます。
2. 「クイズを始める」ボタンを押します。
3. 表示される国旗に対して、正しい国名のボタンを4つの中から選びます（全10問）。
4. 最後にスコアが表示されます。

## 技術スタック
- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (ES6+)
- [REST Countries API](https://restcountries.com/)
